## Technology Contributions to Efficiency

### 1. VDF-Based Consensus (40% efficiency gain)
- **Deterministic computation**: No wasted hash attempts
- **Time-sequential**: Cannot parallelize wastefully
- **Fair enforcement**: Equal opportunity for all miners
- **Energy cost**: ~95 Wh per block

### 2. Proof-of-Work Optimization (30% efficiency gain)
- **Blake3 hash function**: Efficient CPU computation
- **LWMA difficulty adjustment**: Stable hash difficulty
- **Mobile-friendly mining**: 3-5W per device
- **Energy cost**: ~47.5 Wh per block

### 3. Network Efficiency (20% efficiency gain)
- **Lightweight P2P**: Optimized Gossipsub
- **Mobile support**: Reduced bandwidth requirements
- **Compact blocks**: Small header size
- **Energy cost**: ~5 Wh per block

### 4. Mobile Mining (10% efficiency gain)
- **Distributed deployment**: No centralized data centers
- **Battery optimization**: Adjustable intensity
- **Solar/renewable ready**: Low continuous power
- **1 AXM reward**: incentivizes participation
