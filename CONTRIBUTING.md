# Contributing to AXIOM Protocol

Thank you for your interest in contributing! This project is experimental and welcomes educational, research, and improvement-focused contributions.

## Guidelines
- Be respectful and constructive in discussions
- Submit issues for bugs, feature requests, or questions
- Fork the repo and submit pull requests for code changes
- Write clear commit messages and document your changes
- Add tests for new features or bug fixes
- Update documentation as needed

## Code Review
- All PRs require review before merging
- Automated tests must pass
- Major changes should be discussed in issues first

## Getting Started
- See README and TECHNICAL_SPEC.md for project status
- See ROADMAP.md for priorities
- Ask questions in issues if anything is unclear
